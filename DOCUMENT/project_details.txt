================================================================================
                              PROJECT OVERVIEW
================================================================================

Morpheus Echo is a revolutionary anonymous confession social media platform
that allows users to share their deepest truths without ever revealing their
identity. Every user receives a procedurally generated secret identity - a
unique combination of color, creature, location, and number - creating over
2.5 billion possible unique identities.

CORE FEATURES IMPLEMENTED:
✅ Anonymous identity generation with 6 rarity tiers (Mythic 0.1% to Common 60.9%)
✅ Text and voice confessions with 5 voice effects (Normal, Whisper, Deep, Echo, Robotic)
✅ 12 content categories with color-coded pills
✅ 6 reaction types (Me Too, Sending Love, Wow, Same Lol, Stay Strong, Respect)
✅ Real-time anonymous messaging system
✅ Radio mode for passive listening to voice confessions
✅ XP and level progression system (1-300 levels, 6 titles)
✅ Badge system with 7 unique badges
✅ Blur-to-reveal content mechanic
✅ Content moderation with toxicity detection
✅ Emotion detection for feed personalization
✅ Heat score algorithm for trending content
✅ Follow/unfollow system
✅ Notification system with real-time updates
✅ Infinite scroll feed with category filtering
✅ Mobile-first responsive design

================================================================================
                              API ENDPOINTS
================================================================================

AUTHENTICATION:
POST /api/auth/signup - Create new account
POST /api/auth/login - Login existing account

FEED:
GET /api/feed?category=&sort=&page=&limit= - Get confession feed

CONFESSIONS:
POST /api/confessions - Create new confession
GET /api/confessions/:id - Get single confession
POST /api/confessions/:id/react - React to confession
GET /api/confessions/:id/comments - Get comments
POST /api/confessions/:id/comments - Add comment
GET /api/confessions/random - Get random confession (for radio)

USERS:
GET /api/users/:id - Get user profile
GET /api/users/:id/confessions - Get user confessions
POST /api/users/:id/follow - Follow/unfollow user

MESSAGES:
GET /api/messages - Get conversations
GET /api/messages/:userId - Get messages with user
POST /api/messages - Send message

NOTIFICATIONS:
GET /api/notifications - Get notifications
PATCH /api/notifications/read-all - Mark all as read

RADIO:
GET /api/radio?category= - Get radio queue

UPLOAD:
POST /api/upload/audio - Upload audio file

OTHER:
GET /api/categories - Get all categories
GET /api/stats - Get platform stats
GET /health - Health check

================================================================================
                              SOCKET.IO EVENTS
================================================================================

CLIENT EMITS:
- authenticate { userId } - Authenticate socket connection
- send_message { to, content, from } - Send message
- new_reaction { confessionId, authorId, reactionType, from } - Send reaction

SERVER EMITS:
- new_message { from, content, timestamp, messageId } - New message received
- reaction_notification { confessionId, type, from } - Reaction received
- comment_notification { confessionId, commenterName } - Comment received
- follow_notification { followerName } - New follower
- level_up { newLevel, title } - Level up achieved
- notification { type, message, data } - General notification
